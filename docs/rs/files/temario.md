## Objetivo

Esta asignatura tiene como objetivo principal practicar el uso de la tecnología de Bases de Datos en la implementación de sistemas de información, así como familiarizarse con el diseño, implementación y explotación de las mismas.

---

## Software a utilizar

- **Manejador de base de datos:** SQL Server 2022
- **Herramienta CASE:** ER/Studio
- **Diagramador:** DIA
