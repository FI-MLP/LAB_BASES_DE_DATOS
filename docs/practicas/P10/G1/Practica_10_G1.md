### 1.1. Objetivo

El alumno comprenderá y confirmará el uso del lenguaje SQL para realizar la administración de los datos empleando las instrucciones INSERT, UPDATE y DELETE. Comprenderá las principales características de una transacción, comprobará y configurará los diferentes niveles de aislamiento empleados por un RDBMS

### 1.2. Actividades previas

1. Antes de iniciar con la práctica, realizar, revisar y calificar las actividades del documento previo a esta práctica. Realice las actividades del script `10-PREVIO-DML.sql`

---

### 1.3. Actividades a realizar

2. Realice las actividades del script `10_practica_DML.sql`
3. Se requiere registrar una casa de 200 metros cuadrados ubicado en calle Domingo Ramos numero 52 sección 2 fraccionamiento Las américas, latitud 14258, longitud 695841 del Señor Gustavo Meza Salas, su precio de venta de $3,800,000
4. Ahora quieren llevar el control de los adeudos, por lo que le piden implementar en la base de datos (datos_venta) y obtener los adeudos de cada inmueble
5. Ya no se dará el servicio de gas natural y le piden eliminarlo de la BD (no hay borrado en cascada)
6. Insertar 4 pagos para cada venta
7. Insertar 4 precios (históricos precios) para cada inmueble
8. Insertar 3 servicios para cada inmueble vendido
